# Manual Solver Reference

## Fast working checklist

1. restate the objective in one line
2. write down the governing formula or constraint
3. solve the smallest exact subproblem first
4. script arithmetic if manual calculation risks a mistake
5. record the reusable pattern after the answer is done

## Common failure modes

- solving a prettier problem than the prompt asked
- spending too long polishing derivations
- not logging the shortcut after finding it
